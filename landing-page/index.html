Installation Guide

Get started with Deep AI in minutes

1

Install Node.js

Deep AI requires Node.js version 16 or higher.

Download Node.js

npm (Node Package Manager) is included with Node.js installation.

2

Download Deep AI

Download the latest version of Deep AI from our GitHub repository.

Download Deep AI
3

Extract the files

Extract the downloaded zip file to a location of your choice.

Make sure you have sufficient permissions for the directory where you extract the files.

1

Windows Installation

For Windows users, simply double-click the included installation batch file.

Double-click install.bat
2

macOS/Linux Installation

For macOS and Linux users, open a terminal in the project directory and run:

chmod +x install.sh
./install.sh
3

Verification

The installer will automatically check and install all required dependencies.

The installation includes Electron, n8n, Google Cloud Speech API, Google Generative AI (Gemini model), SQLite3, and other required packages.

Important: You don't need to manually install or start n8n - the application handles this automatically!

1

Open Terminal/Command Prompt

Navigate to the project directory in your terminal or command prompt.

cd path/to/deep-ai
2

Install Electron First

Electron is a large download (over 250 MB), so it's recommended to install it first:

node install-electron.js

This may take several minutes due to Electron's large size.

3

Install All Other Dependencies

After Electron is installed, run the dependencies installer script:

node install-dependencies.js

If your install is interrupted or fails, try running the scripts again. A slow or unstable internet connection may cause delays.

1

Start the Application

After installation, run the application using npm:

npm start
2

Configure Settings

On first launch, click the gear icon to access settings.

The application works with n8n in two ways:

  • Integrated Mode: The app will automatically start n8n when launched, and stop it when closed
  • External Mode: If n8n is already running (e.g., you started it separately with npx n8n start), the app will detect and connect to it automatically

The application will automatically use Gemini AI if n8n is not configured or unavailable.

3

Build Standalone Application (Optional)

To create a standalone executable:

npm run build

This will create an installer in the dist folder and a portable executable for distribution.
